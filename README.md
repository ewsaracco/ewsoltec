# EWSolTec - Consultoría Tecnológica

Este repositorio contiene el código fuente de la landing page de **EWSolTec**, mi emprendimiento personal de consultoría y soporte IT enfocado en PyMEs y profesionales.

## 🛠️ Servicios Ofrecidos
La web detalla mis servicios principales:
* **Diagnóstico de Hardware:** Optimización y actualización de estaciones de trabajo.
* **Infraestructura de Redes:** Relevamiento y estabilización de conectividad.
* **Asesoramiento de Software:** Gestión de licencias y herramientas de productividad.
* **Correo y Web:** Configuración de dominios y correos corporativos.

## 💻 Tecnologías Utilizadas
Para mantener la web ligera, rápida y fácil de mantener, utilicé:
* **HTML5:** Estructura semántica.
* **CSS3:** Diseño personalizado con enfoque "Mobile First" (responsivo).
* **Google Fonts:** Tipografía Roboto para una lectura clara.

## 👤 Sobre el Proyecto
Este es un proyecto unipersonal. Busco transmitir transparencia y un trato directo "de igual a igual" con el dueño de la empresa, evitando tecnicismos innecesarios y enfocándome en soluciones prácticas.
